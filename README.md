# PolicyLens

A Chrome extension that analyzes privacy policies using OpenAI and gives them a scored privacy grade.

## Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [npm](https://www.npmjs.com/) v9+
- A Chromium-based browser (Chrome, Edge, Brave, etc.)
- An [OpenAI API key](https://platform.openai.com/api-keys) with access to `gpt-4o-mini`

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Configure your API key

Copy the example env file and fill in your key:

```bash
cp .env.example .env
```

Open `.env` and set your key:

```
VITE_OPENAI_API_KEY=sk-...
```

> `.env` is gitignored and will never be committed.

### 3. Build the extension

```bash
npm run build
```

Output is written to `dist/`.

### 4. Load into Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode** (toggle in the top-right corner)
3. Click **Load unpacked**
4. Select the `dist/` folder inside this project

The PolicyLens icon will appear in your extensions bar.

## Usage

1. Navigate to any privacy policy page (e.g. `https://privacycenter.instagram.com/policy`)
2. A floating **PolicyLens** button will appear on the page
3. Click it — the sidebar opens and the policy is analyzed
4. Results include:
   - A **privacy score** (0–100) and letter grade
   - Per-section summaries with sensitivity ratings
   - A **"How was this score calculated?"** breakdown showing every deduction and bonus applied

## Development

Watch mode — rebuilds on every file save:

```bash
npm run dev
```

After each rebuild, go to `chrome://extensions` and click the **reload** button (↺) on the PolicyLens card to pick up changes.

## Debug logging

To enable verbose console logs, set in `.env`:

```
VITE_DEBUG_POLICYLENS=true
```

Then rebuild. Logs appear in two places:

| Log source                               | Where to find it                                                                 |
| ---------------------------------------- | -------------------------------------------------------------------------------- |
| Content script (`[PolicyLens][content]`) | DevTools of the policy page (`F12` → Console)                                    |
| Service worker (`[PolicyLens][sw]`)      | `chrome://extensions` → PolicyLens → **Inspect views: service worker** → Console |

## Scoring

Scores are computed locally from flags the AI extracts — not generated by the AI itself.

| Deduction                 | Points |
| ------------------------- | ------ |
| Sells user data           | −25    |
| Biometric data collected  | −20    |
| Precise location tracked  | −15    |
| Health data collected     | −15    |
| No data deletion option   | −15    |
| Third-party advertising   | −10    |
| Indefinite data retention | −10    |
| Children's data collected | −10    |

| Bonus                | Points |
| -------------------- | ------ |
| Right to delete      | +10    |
| Data not sold        | +10    |
| Encryption mentioned | +5     |
| Opt-out available    | +5     |

Formula: `Score = 100 − penalties + bonuses` (penalties capped at 80, final score clamped to 0–100).
